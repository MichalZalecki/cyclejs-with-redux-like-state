# cyclejs-with-redux-like-state

## Node

```
6.0.0
```

## Scripts

### Start **Express** (for production)

```bash
npm start
```

### Start **WebpackDevServer** (for development)

```bash
npm run start:dev
```

### Build (also run in *postinstall*)

```bash
npm run build
```

### Unit tests

```bash
npm run test
```

or

```bash
npm run test:watch
```

### ESLint

```bash
npm run lint
```
